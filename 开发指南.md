# 金融贷款 BI Dashboard 开发说明

（用于指导 VSCode Copilot 自动生成代码）

## 一、项目目标

开发一个 **金融贷款 BI 可视化系统**，用于展示贷款业务指标趋势。

系统通过调用 API 获取指标数据，并使用 **Apache ECharts** 在前端绘制 BI 图表。

第一阶段只实现：

* API 数据获取
* 数据解析
* 折线图展示
* BI Dashboard 页面

数据接口：

```
https://atlas.asksquirrel.ai/analyst/api/v1/excel/metrics
```

API 返回数据格式如下：

```json
{
  "metrics": [
    "Annual_Inc",
    "APR",
    "Avg_Orig_Bal",
    "Debt_Consolidation_Pct",
    "DTI",
    "Existing_Borrower_Pct",
    "FICO",
    "Grade_1_Pct",
    "IR",
    "Orig_Bal",
    "PTI",
    "Term"
  ],
  "grouped": {
    "Annual_Inc": [
      {
        "vintage_month": "2016-01-31",
        "Metrics": "Annual_Inc",
        "Value": 82859.717
      },
      {
        "vintage_month": "2016-02-29",
        "Metrics": "Annual_Inc",
        "Value": 85455.5655
      },
      {
        "vintage_month": "2016-03-31",
        "Metrics": "Annual_Inc",
        "Value": 84644.2914
      }
    ]
  }
}
```

# 五、数据字段说明

### metrics

指标列表：

```
Annual_Inc
APR
Avg_Orig_Bal
Debt_Consolidation_Pct
DTI
Existing_Borrower_Pct
FICO
Grade_1_Pct
IR
Orig_Bal
PTI
Term
```

---

### grouped

每个指标对应一组时间序列数据。

结构：

```
grouped[metricName] -> array
```

数组元素：

```
vintage_month   时间（月）
Metrics         指标名称
Value           指标值
```

# 六、TypeScript 类型定义

```

```

```

```

结构：

```
vintage_month: string
Metrics: string
Value: number
```

API 返回类型：

```
MetricsResponse
```

结构：

```
metrics: string[]

grouped: {
  [metricName: string]: MetricPoint[]
}
```

# 八、数据转换逻辑

ECharts 需要：

```
xAxis
series
```

但 API 返回是：

```
时间 + 指标值
```

需要转换。

创建：

```
utils/transform.ts
```

函数：

```
transformMetricToChartData()
```

输入：

```
MetricPoint[]
```

输出：

```
{
  dates: string[]
  values: number[]
}
```

转换逻辑：

```
dates  = vintage_month
values = Value
```
